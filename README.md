# DS_intro

## Чему я научился

### Hard Skills (работа с данными)

**1. Подготовка и исследование данных (EDA)**
- Научился загружать данные из разных источников (CSV, SQL, API)
- Проводить первичный анализ: типы данных, пропуски, дубликаты, выбросы
- Визуализировать распределения и зависимости (гистограммы, boxplot, scatter, heatmap)

**2. Статистический анализ**
- Проверять статистические гипотезы
  
**3. Машинное обучение**
- **Регрессия:** прогнозирование числовых значений (Linear Regression, Ridge, Lasso)
- **Классификация:** бинарная и многоклассовая (Logistic Regression, Decision Tree, Random Forest)
- **Оценка моделей:** метрики MAE, MSE, RMSE, R², Accuracy, Precision, Recall, F1, ROC-AUC
- **Борьба с переобучением:** кросс-валидация, регуляризация, подбор гиперпараметров (GridSearchCV)

**4. Инструменты и практики**
- Python (основные библиотеки)
- SQL (JOIN, GROUP BY, оконные функции, подзапросы)
- Git (коммиты, ветки, pull request'ы, работа в команде)
